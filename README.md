## E4X detector

Searches a Firefox addon XPI or JS source file for uses of E4X.

## Binaries

  * `bin/e4x-detector SHELL INPUT` : search INPUT file for E4X using the SpiderMonkey executable SHELL

## Functions

  * `report(shell, path)` : run the search
